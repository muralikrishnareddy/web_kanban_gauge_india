Sales Team Invoice Amount in India Format in Kanban View with Decimals

To show Sales Team Invoice Target and Invoiced this Month with Crores/Lakhs/Thousnads instead of Millions in Kanban View.

168.8 Crores showing as 168.8C with this moodule where as Odoo by default shows as 2G

and 200k as 2Lakhs(2L) and 223M as 22.3 Crores (22.3C)

45.7 Thousand Crores showing as 45.7KC with this moodule where as Odoo by default shows as 457G

and 100k as 1Lakh(1L) and 3M as 25.4L (which is correct amount with out rounding)

By Default, Decimals will be zero, to show Decimals Configure from

Company --> Configuration tab --> Sales Team group Digits box.
